Configurable fixes and patches for Tetote Connect.

I/O can be found here: https://github.com/akechi-haruka/ttio

---------------------------------------

Requires BepInEx 5.

Features (everything configurable):

New:
* Change default/error language
* Force usage of aeroTap or Intel RealSense
* Extra I/O bindings (in addition to ttio)
* Allow maximum dancer height to go past 175 and recalculate note positions
* Adjust game position for non-standard monitor stands

Fixes:
* Disable event ending dates
* Add prices to shop items
* Fix inaccessible information menu

Home Usage:
* Disable automatic screen rotation
* Redirect game NVRAM
* Soften touchscreen check
* Disable camera startup check
